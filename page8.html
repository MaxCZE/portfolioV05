Digitalisation du Suivi Des Collaborateurs

Le fonctionnement du projet

Dans l'entreprise, chaque année, les salariés passent un entretien annuel d'évaluation. Avant de commencer une mission chez un client, ils ont des entretiens professionnels. Lors de leurs missions chez le client, ils ont régulièrement des suivis de missions. Dans le but de faciliter la gestion de ces trois documents, l'entreprise a lancé un projet interne. Grâce aux compétences acquises en développement web lors de la formation Nextoo Academy, les stagiaires deviennent alors capables de développer de nouvelles fonctionnalités sur ce projet ou de corriger des bugs existants.

Grâce à son interface user friendly, les salariés de l'entreprise peuvent modifier, réutiliser, partager, remplir et supprimer les documents administratif dans une plateforme commune.

En appliquant les règles de la métrologie agile (daily, user stories, rôles...) nous étions en mesure de progresser sur le projet en évitant un maximum les incohérences entre les versions de code des différents développeurs.

Ce projet est le premier projet professionnel auquel j'ai contribué.

Compétences techniques acquises :
    - développement front en Vuejs
    - développement back en spring-boot
    - découverte de l'IDE Intellij
    - répartition des différentes tâches avec les USs jira

Savoir-être acquis : 
    - coopération
    - attention envers le besoin client (ici besoin des collaborateurs)
    - autonomie
    - spontanéité
